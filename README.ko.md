# 3 min. Calendar

**바쁜 자영업자를 위한 시간 절약**

단 3분 만에 이번 달 영업 캘린더를 완성하세요.

Instagram이나 LINE으로 고객에게 바로 공유할 수 있습니다.

---

## 이 앱을 만든 이유

"가게 영업 캘린더를 Instagram에 올리고 싶다."

검색해보니 나오는 건 **매달 Canva 템플릿을 편집하는 방법**뿐이었습니다.

- 표 선이 어긋나서 위치를 수동으로 조정해야 함
- 공휴일도 매달 수동으로 빨간색으로 바꿔야 함
- 같은 작업을 매달 반복...

**"이렇게 번거로운 일이 있나?"**

하지만 전용 앱이 없었습니다.

그래서 만들었습니다.

버튼만 누르면 됩니다. 공휴일은 자동. 3분이면 완성.

바쁜 일상을 보내는 자영업자분들에게 조금이라도 시간을 돌려드릴 수 있으면 좋겠습니다.

## 사용법 (간단 3단계)

### 1단계: 날짜를 탭하여 입력

캘린더의 날짜를 탭하면 아래에 편집 화면이 나타납니다.

- **"휴무"** → 정기휴일
- **"◯"** → 여유 있음
- **"△"** → 잔여 소수
- **"✕"** → 예약됨
- **"만석"** → 만석/만실

버튼만 누르면 됩니다!
시간이나 자유로운 코멘트도 입력할 수 있습니다.

### 2단계: 상호명 설정 (선택사항)

오른쪽 상단의 **⚙️설정** 버튼에서 가게 이름과 로고를 설정할 수 있습니다.
캘린더 이미지 왼쪽 상단에 표시됩니다.

### 3단계: 공유 또는 저장

- **📤 공유** → Instagram, LINE, X 등에 바로 게시
- **💾 저장** → 스마트폰이나 컴퓨터에 이미지 저장

**※ 매번 "화면 공유" 허가가 필요합니다**

"공유" 또는 "저장" 버튼을 누르면 "화면을 공유하시겠습니까?"라는 확인이 나타납니다.

이것은 캘린더를 이미지로 저장하기 위해 필요한 작업입니다.
"허용"을 누르세요.

(보안상의 이유로 매번 확인이 표시됩니다)

오른쪽 상단의 **⚙️설정** 버튼을 누르면 다음 설정이 가능합니다:

| 설정 항목   | 설명                                  |
| ----------- | ------------------------------------- |

| 언어        | 日本語 / English / 中文 / 한국어 등   |
| 주 시작일   | 일요일 시작 / 월요일 시작             |
| 공휴일 표시 | 켜기/끄기                             |
| 국가/지역   | 거주 국가를 선택하면 공휴일 자동 색상 |
| 육요 표시   | 다이안・붓스메츠 등을 날짜에 표시     |

| 데이터 관리 | 입력한 데이터의 백업・복원            |

## 테마 (색상 패턴)

캘린더 아래의 색상 버튼으로 디자인을 선택할 수 있습니다.

**14가지** 컬러 바리에이션:

- 라이트 계열 (흰색 배경): 기본, 빨강, 주황, 노랑, 초록, 파랑, 보라
- 다크 계열 (검정 배경): 기본, 빨강, 주황, 노랑, 초록, 파랑, 보라

월별로 다른 테마를 설정할 수도 있습니다.
계절에 맞춰 색상을 바꿔보세요.

## QR 코드 생성 기능

화면 상단의 "QR" 탭을 누르면 QR 코드 생성 화면이 열립니다.

- 가게 홈페이지나 SNS URL 입력
- QR 코드 디자인 커스터마이징 (색상, 모양, 로고 삽입)
- 생성된 QR 코드 다운로드

### QR 코드 기능을 넣은 이유

무료 QR 코드 생성 사이트 중에는 수상한 광고 사이트나 전송 서비스를 경유하는 곳이 있습니다.

**실제로 모르고 그런 QR 코드를 고객에게 나눠주는 사장님들이 있습니다.**

지금 바로 가게에서 사용 중인 QR 코드를 본인 스마트폰으로 읽어보세요.

가게 URL이 아니라 낯선 단축 URL 등이 표시된다면... 그것은 사기 사이트에서 만든 QR 코드입니다.

이 앱으로 다시 만드는 것을 권장합니다.

이 앱의 QR 코드 기능은 **완전 오프라인**으로 작동합니다.
주식회사 덴소웨이브 공식과 마찬가지로, 입력된 URL을 일절 가공하지 않고 그대로 QR 코드로 만듭니다.

안심하고 사용하세요.

## 스마트폰에 설치

이 앱은 PWA(Progressive Web App)를 지원합니다.

스마트폰 홈 화면에 추가하면 일반 앱처럼 사용할 수 있습니다.

### Android의 경우 (Chrome)

1. Chrome에서 이 앱 열기
2. 오른쪽 상단의 "︙" 메뉴 누르기
3. "홈 화면에 추가" 또는 "앱 설치" 누르기
4. 홈 화면에 아이콘 추가됨

### iPhone의 경우 (Safari)

1. Safari에서 이 앱 열기
2. 하단의 "공유" 버튼 (□에서 ↑가 나오는 아이콘) 누르기
3. "홈 화면에 추가" 누르기
4. 홈 화면에 아이콘 추가됨

설치 후에는 오프라인에서도 사용할 수 있습니다.

## 자주 묻는 질문

### Q. 인터넷 없이도 사용할 수 있나요?

**네, 사용할 수 있습니다.**
한 번 페이지를 열면 오프라인에서도 작동합니다.
입력한 데이터는 스마트폰(또는 컴퓨터)에 저장됩니다.

### Q. 만든 캘린더는 어디에 저장되나요?

사용 중인 브라우저(Chrome, Safari 등)에 자동 저장됩니다.
앱을 닫아도 다음에 열면 그대로 이어서 편집할 수 있습니다.

### Q. 다른 사람이 볼 수 있나요?

**없습니다.**
데이터는 본인의 스마트폰(또는 컴퓨터)에만 저장됩니다.
인터넷에 업로드되지 않습니다.

### Q. 기기 변경하면 데이터가 사라지나요?

**네, 사라집니다.**
새 스마트폰에는 자동으로 이전되지 않습니다.

데이터를 이전하려면 설정 화면의 "데이터 관리"에서 "내보내기"로 백업 파일을 저장하고, 새 스마트폰에서 "가져오기"하세요.

### Q. "화면 공유"가 뭔가요? 무섭습니다

안심하세요. 이것은 캘린더를 이미지로 저장하기 위한 기능입니다.

본인의 화면이 누군가에게 보이는 것이 아닙니다.
"허용"을 누르면 캘린더 부분만 이미지로 잘라서 저장합니다.

### Q. Instagram이나 LINE에 게시하는 방법을 모르겠습니다

1. **📤공유** 버튼 누르기
2. "화면 공유" 허가 누르기
3. "Instagram"이나 "LINE" 선택
4. 평소처럼 게시

잘 안 되면 **💾저장**으로 이미지를 저장한 후,
평소 사진을 게시하는 것과 같은 방법으로 게시하세요.

### Q. 육요가 뭔가요?

"다이안", "붓스메츠" 등을 말합니다.
일본에서는 결혼식은 다이안이 좋고, 장례식은 토모비키를 피하는 등 길흉을 신경 쓰는 분들이 있습니다.

음식점이나 미용실에서 고객에게 "오늘은 다이안입니다"라고 알리고 싶을 때 편리합니다.
필요 없는 분은 끄세요.

## 문제가 있을 때

잘 작동하지 않으면 다음을 시도해 보세요:

1. **브라우저를 최신 버전으로 업데이트**
   - iPhone: 설정 → 일반 → 소프트웨어 업데이트
   - Android: Google Play 스토어에서 브라우저 업데이트

2. **페이지 새로고침**
   - 화면을 아래로 당겼다 놓기
   - 또는 주소창 옆의 새로고침 버튼 누르기

3. **다른 브라우저로 시도**
   - Chrome, Safari, Edge 등

4. **"화면 공유"가 나오지 않는 경우**
   - 브라우저 설정에서 "화면 공유"가 허용되어 있는지 확인하세요
   - 일부 오래된 브라우저에서는 지원하지 않을 수 있습니다
