# 3 min. Calendar

**すべての頑張る個人経営者の味方**

カップラーメンができる3分で、今月の営業カレンダーが完成。
InstagramやLINEで、お客様にサッと共有できます。

---

## こんな方におすすめ

- 飲食店、カフェ、居酒屋のオーナーさん
- 旅館、民宿、ゲストハウスの経営者さん
- 美容室、ネイルサロン、エステの店長さん
- 教室、スタジオ、ジムのインストラクターさん
- その他、定休日や予約状況をお知らせしたいすべての方

---

## 使い方（かんたん3ステップ）

### ステップ1：日付をタップして入力

カレンダーの下に、その月の日付が並んでいます。

- **「休」** → 定休日
- **「◯」** → 空きあり
- **「△」** → 残りわずか
- **「✕」** → 予約済み
- **「満」** → 満席・満室

ボタンをポンポン押すだけ！
もちろん「10:00-18:00」のように自由に入力もできます。

### ステップ2：店名を設定（お好みで）

右上の **⚙️設定** ボタンから、お店の名前を入力できます。
カレンダー画像に表示されます。

### ステップ3：シェアまたは保存

- **📤 シェア** → Instagram、LINE、Xなどに直接投稿
- **💾 保存** → スマホやパソコンに画像を保存

---

## 設定でできること

右上の **⚙️設定** ボタンを押すと、以下の設定ができます：

| 設定項目   | 説明                                     |
| ---------- | ---------------------------------------- |
| 言語       | 日本語 / English                         |
| 国/地域    | お住まいの国を選ぶと、祝日が自動で色付き |
| 祝日を表示 | オン/オフを切り替え                      |
| テーマ     | カレンダーの色を6種類から選べます        |
| 店名       | カレンダー上部に表示されるお店の名前     |
| 週の開始日 | 日曜始まり / 月曜始まり                  |

---

## テーマ（色のパターン）

お店の雰囲気に合わせて、6種類から選べます：

- **ダーク** - 落ち着いた黒背景
- **ライト** - 明るい白背景
- **カフェ** - 温かみのある茶色
- **ネイチャー** - 自然な緑
- **オーシャン** - 爽やかな青
- **サンセット** - 暖かいオレンジ

## よくある質問

### Q. インターネットがなくても使えますか？

**はい、使えます。**
一度ページを開けば、オフラインでも動作します。
入力したデータはスマホ（またはパソコン）の中に保存されます。

### Q. 作ったカレンダーはどこに保存されますか？

お使いのブラウザ（Chrome、Safari等）の中に自動保存されます。
アプリを閉じても、次に開いたときにそのまま続きから編集できます。

### Q. 他の人に見られることはありますか？

**ありません。**
データはあなたのスマホ（またはパソコン）の中だけに保存されます。
インターネット上にアップロードされることはありません。

### Q. 機種変更したらデータは消えますか？

**はい、消えます。**
新しいスマホには引き継がれません。
大切なカレンダーは画像として保存しておいてください。

### Q. InstagramやLINEへの投稿方法がわかりません

1. **📤シェア** ボタンを押す
2. 「Instagram」や「LINE」を選ぶ
3. いつも通り投稿する

もしうまくいかない場合は、**💾保存** で画像を保存してから、
いつも写真を投稿するのと同じ方法で投稿してください。

## 困ったときは

うまく動かない場合は、以下をお試しください：

1. **ブラウザを最新版にする**
   - iPhoneの方：設定 → 一般 → ソフトウェアアップデート
   - Androidの方：Google Playストアでブラウザを更新

2. **ページを再読み込みする**
   - 画面を下に引っ張って離す
   - または、アドレスバーの横の更新ボタンを押す

3. **別のブラウザで試す**
   - Chrome、Safari、Edgeなど

## このアプリを作った理由

「お店の営業カレンダーをInstagramに載せたい」

そう思って調べたら、出てくるのは**Canvaでテンプレートを毎月編集する方法**ばかり。

- 罫線がずれる、位置を手動で調整
- 祝日も毎月手動で赤くする
- 同じ作業を毎月繰り返し…

**「こんな面倒なことがあるか？」**

でも、専用のアプリがほとんどなかったんです。

だから作りました。

ボタンをポンポン押すだけ。祝日は自動。3分で完成。

忙しい毎日を送る個人経営者の方に、少しでも時間を返せたら嬉しいです。
