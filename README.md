**がんばる個人経営者の時短に**

カップラーメンができる3分で、今月の営業カレンダーが完成。

InstagramやLINEで、お客様にサッと共有できます。

---

## このアプリを作った理由

「お店の営業カレンダーをInstagramに載せたい」

そう思って調べたら、出てくるのは**Canvaでテンプレートを毎月編集する方法**ばかり。

- 罫線がずれる、位置を手動で調整
- 祝日も毎月手動で赤くする
- 同じ作業を毎月繰り返し…

**「こんな面倒なことがあるか？」**

でも、専用のアプリがなかったんです。

だから作りました。

ボタンをポンポン押すだけ。祝日は自動。3分で完成。

忙しい毎日を送る個人経営者の方に、少しでも時間を返せたら嬉しいです。

## 使い方（かんたん3ステップ）

### ステップ1：日付をタップして入力

カレンダーの日付をタップすると、下に編集画面が出てきます。

- **「休」** → 定休日
- **「◯」** → 空きあり
- **「△」** → 残りわずか
- **「✕」** → 予約済み
- **「満」** → 満席・満室

ボタンをポンポン押すだけ！
時間や自由なコメントも入力できます。

### ステップ2：店名を設定（お好みで）

右上の **⚙️設定** ボタンから、お店の名前やロゴを設定できます。
カレンダー画像の左上に表示されます。

### ステップ3：シェアまたは保存

- **📤 シェア** → Instagram、LINE、Xなどに直接投稿
- **💾 保存** → スマホやパソコンに画像を保存

**※ 毎回「画面の共有」の許可が必要です**

「シェア」または「保存」ボタンを押すと、「画面を共有しますか？」という確認が出ます。

これはカレンダーを画像として保存するために必要な操作です。
「許可」を押してください。

（セキュリティのため、毎回確認が表示されます）

## 設定でできること

右上の **⚙️設定** ボタンを押すと、以下の設定ができます：

| 設定項目     | 説明                                     |
| ------------ | ---------------------------------------- |
| アプリの外観 | ライト / ダーク                          |
| 言語         | 日本語 / English                         |
| 週の開始日   | 日曜始まり / 月曜始まり                  |
| 祝日を表示   | オン/オフを切り替え                      |
| 国/地域      | お住まいの国を選ぶと、祝日が自動で色付き |
| 六曜を表示   | 大安・仏滅などを日付に表示（飲食店向け） |
| 和暦を使用   | 「2025年」を「令和7年」で表示            |
| 店名         | カレンダー左上に表示されるお店の名前     |
| 店名ロゴ     | 店名の横に表示される小さなロゴ画像       |
| 背景画像     | カレンダーの背景に好きな画像を設定       |
| データ管理   | 入力したデータのバックアップ・復元       |

## テーマ（色のパターン）

カレンダーの下にある色のボタンで、デザインを選べます。

**14種類**のカラーバリエーション：

- ライト系（白背景）: 標準、赤、オレンジ、黄、緑、青、紫
- ダーク系（黒背景）: 標準、赤、オレンジ、黄、緑、青、紫

月ごとに違うテーマを設定することもできます。
季節に合わせて色を変えてみてください。

## QRコード作成機能

画面上部の「QR」タブを押すと、QRコード作成画面が開きます。

- お店のホームページやSNSのURLを入力
- QRコードのデザインをカスタマイズ（色、形、ロゴ挿入）
- 作成したQRコードをダウンロード

### なぜQRコード機能を付けたのか

無料のQRコード作成サイトの中には、怪しい広告サイトや転送サービスを経由させるものがあります。

**実際に、知らずにそういったQRコードをお客様に配ってしまっている店主さんがいます。**

いま一度、お店で使っているQRコードをご自分のスマホで読み取ってみてください。

お店のURLではなく、見知らぬ短縮URLなどが表示された場合…それは詐欺サイトで作成されたQRコードです。

このアプリでの再作成をおすすめします。

このアプリのQRコード機能は**完全オフライン**で動作します。
株式会社デンソーウェーブ公式と同じく、入力されたURLを一切加工せず、そのままQRコードにしています。

安心してお使いください。

## スマホにインストール

このアプリはPWA（Progressive Web App）に対応しています。

スマホのホーム画面に追加すると、普通のアプリのように使えます。

### Androidの場合（Chrome）

1. Chromeでこのアプリを開く
2. 右上の「︙」メニューを押す
3. 「ホーム画面に追加」または「アプリをインストール」を押す
4. ホーム画面にアイコンが追加される

### iPhoneの場合（Safari）

1. Safariでこのアプリを開く
2. 下の「共有」ボタン（□から↑が出ているアイコン）を押す
3. 「ホーム画面に追加」を押す
4. ホーム画面にアイコンが追加される

インストール後はオフラインでも使えます。

## よくある質問

### Q. インターネットがなくても使えますか？

**はい、使えます。**
一度ページを開けば、オフラインでも動作します。
入力したデータはスマホ（またはパソコン）の中に保存されます。

### Q. 作ったカレンダーはどこに保存されますか？

お使いのブラウザ（Chrome、Safari等）の中に自動保存されます。
アプリを閉じても、次に開いたときにそのまま続きから編集できます。

### Q. 他の人に見られることはありますか？

**ありません。**
データはあなたのスマホ（またはパソコン）の中だけに保存されます。
インターネット上にアップロードされることはありません。

### Q. 機種変更したらデータは消えますか？

**はい、消えます。**
新しいスマホには自動で引き継がれません。

データを引き継ぎたい場合は、設定画面の「データ管理」から「エクスポート」でバックアップファイルを保存し、新しいスマホで「インポート」してください。

### Q. 「画面の共有」って何ですか？怖いです

ご安心ください。これはカレンダーを画像として保存するための機能です。

あなたの画面が誰かに見られるわけではありません。
「許可」を押すと、カレンダー部分だけを画像として切り取って保存します。

### Q. InstagramやLINEへの投稿方法がわかりません

1. **📤シェア** ボタンを押す
2. 「画面を共有」の許可を押す
3. 「Instagram」や「LINE」を選ぶ
4. いつも通り投稿する

もしうまくいかない場合は、**💾保存** で画像を保存してから、
いつも写真を投稿するのと同じ方法で投稿してください。

### Q. 六曜って何ですか？

「大安」「仏滅」などのことです。
日本では、結婚式は大安がよい、葬式は友引を避けるなど、縁起を気にする方がいます。

飲食店や美容室で、お客様に「大安ですよ」とお伝えしたい場合に便利です。
必要ない方はオフにしてください。

## 困ったときは

うまく動かない場合は、以下をお試しください：

1. **ブラウザを最新版にする**
   - iPhoneの方：設定 → 一般 → ソフトウェアアップデート
   - Androidの方：Google Playストアでブラウザを更新

2. **ページを再読み込みする**
   - 画面を下に引っ張って離す
   - または、アドレスバーの横の更新ボタンを押す

3. **別のブラウザで試す**
   - Chrome、Safari、Edgeなど

4. **「画面の共有」が出ない場合**
   - ブラウザの設定で「画面の共有」が許可されているか確認してください
   - 一部の古いブラウザでは対応していない場合があります
