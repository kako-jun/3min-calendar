## QR Code Creation Feature

I-tap ang "QR" tab sa itaas ng screen para buksan ang QR code creation page.

- I-input ang website o social media URL ng tindahan
- I-customize ang QR code design (kulay, hugis, logo)
- I-download ang nagawang QR code

### Bakit Idinagdag ang QR Code Feature

Ang ibang libreng QR code creation site ay dumadaan sa mga kahina-hinalang ad site o forwarding service.

**Sa katunayan, maraming may-ari ng tindahan ang hindi nakakaalam na nagbibigay sila ng ganyang QR code sa mga customer.**

I-scan ngayon ang QR code na ginagamit sa tindahan mo gamit ang iyong phone.

Kung ang lumabas ay hindi ang URL ng tindahan mo kundi kakaibang short URL... ginawa iyan sa scam site.

Inirerekomenda kong gumawa muli gamit ang app na ito.

Ang QR code feature ng app na ito ay gumagana **nang fully offline**.
Katulad ng DENSO WAVE official, ang na-input na URL ay direktang ginagawang QR code nang walang anumang pagbabago.

Gamitin nang may katiwasayan.

## Mga Madalas Itanong

### Q. Magagamit ba kahit walang internet?

**Oo, magagamit.**
Kapag nabuksan ang page nang isang beses, gagana ito offline.
Ang na-input na data ay naka-save sa phone (o computer).

### Q. Saan naka-save ang ginawang calendar?

Auto-save sa browser na ginagamit (Chrome, Safari, atbp.).
Kahit isara ang app, kapag binuksan muli ay maaaring ipagpatuloy ang pag-edit.

### Q. Makikita ba ng iba?

**Hindi.**
Ang data ay naka-save lang sa phone (o computer) mo.
Hindi ito ina-upload sa internet.

### Q. Mawawala ba ang data kapag nagpalit ng phone?

**Oo, mawawala.**
Hindi automatic na naililipat sa bagong phone.

Para ilipat ang data, pumunta sa "Data Management" sa settings screen â†’ "Export" para i-save ang backup file, pagkatapos "Import" sa bagong phone.
